# Get Project Info

Retrieve detailed information about your OpenFiles project, including usage statistics, settings, and metadata. This endpoint requires JWT authentication.

## Authentication Required

This endpoint requires JWT authentication (not API key). Use your GitHub-authenticated session token.

## Use Cases

- **Usage monitoring** - Check storage usage and file counts
- **Project settings** - Retrieve project configuration
- **Dashboard data** - Display project information in your app
- **Billing information** - Check usage against quotas

## Example Usage

<CodeGroup>
```javascript JavaScript
const response = await fetch('https://api.openfiles.com/project', {
  headers: {
    'Authorization': `Bearer ${JWT_TOKEN}`,
    'Content-Type': 'application/json'
  }
})

const project = await response.json()
console.log(project.name, project.fileCount)
```

```python Python
import requests

response = requests.get(
    'https://api.openfiles.com/project',
    headers={'Authorization': f'Bearer {JWT_TOKEN}'}
)

project = response.json()
print(project['name'], project['file_count'])
```

```bash cURL
curl 'https://api.openfiles.com/project' \
  -H 'Authorization: Bearer YOUR_JWT_TOKEN'
```
</CodeGroup>
