# AI Framework Tools

Use OpenFiles with any AI provider that supports tool calling - Anthropic Claude, Google Gemini, Cohere, and more.

## How It Works

1. **Import the tools** - Get OpenFiles tool definitions
2. **Add to your AI** - Include tools in your AI requests
3. **Process responses** - Handle file operations automatically
4. **Continue conversation** - AI gets results and responds naturally

## Features

- ‚úÖ **Universal Compatibility** - Works with any AI framework
- ‚úÖ **Standard Tool Format** - OpenAI-compatible tool definitions
- ‚úÖ **Selective Processing** - Only handles OpenFiles tools
- ‚úÖ **Automatic Execution** - File operations happen seamlessly
- ‚úÖ **Rich Error Handling** - Comprehensive error management

## Framework Examples

### Anthropic Claude

<CodeGroup>
```typescript TypeScript
import { OpenFilesTools } from '@openfiles/sdk/tools'
import Anthropic from '@anthropic-ai/sdk'

const tools = new OpenFilesTools({ 
  apiKey: process.env.OPENFILES_API_KEY 
})
const anthropic = new Anthropic({ 
  apiKey: process.env.ANTHROPIC_API_KEY 
})

// Add OpenFiles tools to Claude
const response = await anthropic.messages.create({
  model: 'claude-3-5-sonnet-20241022',
  max_tokens: 1024,
  messages: [{
    role: 'user',
    content: 'Create a vacation itinerary for 5 days in Paris and save it as paris-trip.md'
  }],
  tools: tools.definitions
})

// Process file operations
const processed = await tools.processToolCalls(response)
if (processed.handled) {
  console.log(`‚úÖ Handled ${processed.results.length} file operations`)
}
```

```python Python
from openfiles.tools import OpenFilesTools
import anthropic

tools = OpenFilesTools(api_key=os.getenv('OPENFILES_API_KEY'))
client = anthropic.Anthropic(api_key=os.getenv('ANTHROPIC_API_KEY'))

# Add OpenFiles tools to Claude
response = client.messages.create(
    model='claude-3-5-sonnet-20241022',
    max_tokens=1024,
    messages=[{
        'role': 'user',
        'content': 'Create a vacation itinerary for 5 days in Paris and save it as paris-trip.md'
    }],
    tools=tools.definitions
)

# Process file operations
processed = await tools.process_tool_calls(response)
if processed.handled:
    print(f'‚úÖ Handled {len(processed.results)} file operations')
```
</CodeGroup>

### Google Gemini

<CodeGroup>
```typescript TypeScript
import { OpenFilesTools } from '@openfiles/sdk/tools'
import { GoogleGenerativeAI } from '@google/generative-ai'

const tools = new OpenFilesTools({ 
  apiKey: process.env.OPENFILES_API_KEY 
})
const genai = new GoogleGenerativeAI(process.env.GOOGLE_AI_API_KEY)

// Add OpenFiles tools to Gemini
const model = genai.getGenerativeModel({
  model: 'gemini-1.5-pro',
  tools: [{ functionDeclarations: tools.definitions }]
})

const response = await model.generateContent({
  contents: [{
    role: 'user',
    parts: [{ text: 'Write a meal plan for the week with healthy recipes and save it as meal-plan.md' }]
  }]
})

// Process file operations
const processed = await tools.processToolCalls(response)
```

```python Python
from openfiles.tools import OpenFilesTools
import google.generativeai as genai

tools = OpenFilesTools(api_key=os.getenv('OPENFILES_API_KEY'))
genai.configure(api_key=os.getenv('GOOGLE_AI_API_KEY'))

# Add OpenFiles tools to Gemini
model = genai.GenerativeModel(
    model_name='gemini-1.5-pro',
    tools=[{'function_declarations': tools.definitions}]
)

response = model.generate_content(
    'Write a meal plan for the week with healthy recipes and save it as meal-plan.md'
)

# Process file operations
processed = await tools.process_tool_calls(response)
```
</CodeGroup>

## Available Tools

| Tool | Description | Use Case |
|------|-------------|----------|
| `write_file` | Create new file | AI generates shopping lists, itineraries, recipes |
| `read_file` | Read and display file | AI reviews existing content before making changes |
| `edit_file` | Modify specific text | AI fixes typos, updates lists, adds items |
| `list_files` | Browse directory | AI explores saved files to find what you need |
| `append_to_file` | Add content to end | AI adds new items to lists, notes to journals |
| `overwrite_file` | Replace entire content | AI completely rewrites outdated documents |
| `get_file_metadata` | Get file info only | AI checks file size, version, dates |
| `get_file_versions` | Access file history | AI reviews changes over time or reverts to previous versions |

## Error Handling

<CodeGroup>
```typescript TypeScript
try {
  const processed = await tools.processToolCalls(response)
  
  for (const result of processed.results) {
    if (result.status === 'success') {
      console.log(`‚úÖ ${result.function}: ${result.data?.path}`)
    } else {
      console.error(`‚ùå ${result.function}: ${result.error}`)
    }
  }
} catch (error) {
  console.error('Tool processing failed:', error.message)
}
```

```python Python
try:
    processed = await tools.process_tool_calls(response)
    
    for result in processed.results:
        if result.status == 'success':
            print(f'‚úÖ {result.function}: {result.data.path if result.data else "completed"}')
        else:
            print(f'‚ùå {result.function}: {result.error}')
            
except Exception as error:
    print(f'Tool processing failed: {error}')
```
